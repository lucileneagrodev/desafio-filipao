Desafio Classificador de nível de Herói

Projeto de desafio da Digital Innovation One (DIO),utilizando javaScript.

Foi utilizado:

- Variáveis;
- Operadores;
- Laços de repetição;
- Estruturas de decisões.

Objetivo

Criar uma variável para armazenar o nome e a quantidade de experiência (XP) de um herói, depois utilizar uma estrutura de decisão para fornecer uma mensagem
informando o nome do Herói e o nível dele.
